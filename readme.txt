== Description ==
Plugin „Ubytování“ je určen pro správu rezervací a ubytovaných hostů v malých penzionech, apartmánech nebo chalupách. Obsahuje přehledný kalendář, možnost ruční i online rezervace, evidenci hostů, generování faktur, blokaci termínů a další funkce.

== Installation ==
1. Nahrajte plugin do adresáře `/wp-content/plugins/`.
2. Aktivujte plugin přes menu „Pluginy“ v administraci WordPressu.
3. V administraci nastavte počet pokojů, ceny a případnou aktivaci licence.

== Frequently Asked Questions ==

= Je plugin zdarma? =
Základní funkce jsou zdarma pro jeden pokoj. Pro více pokojů a plnou funkčnost je nutná licence.

= K čemu slouží blokace termínů? =
Pomocí blokace lze uzavřít libovolný termín bez vytvoření rezervace – např. kvůli rekonstrukci, soukromému využití nebo technické odstávce.

= Je plugin kompatibilní s mobilními zařízeními? =
Ano, kalendář i formuláře jsou plně responzivní.
